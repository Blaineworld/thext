Thext's Server Commands

This file details the various commands that clients send to servers to load text lines, open files, and other actions.
See this document for the commands that servers send to clients.

  1. Authentication
  2. Editing

Authentication

auth.login string:username [string:password]
…attempts to authenticate the user.
auth.logout
…attempts to (unauthenticate?) the user.
auth.register string:username [string:password]
…requests that a new user account is created.

Editing

edit.get int64:first [int64:last]
…requests the content of one or more lines.
edit.set int64:position string:content
…replaces the content of a line.
