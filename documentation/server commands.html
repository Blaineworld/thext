Thext's Server Commands

This file details the various commands clients send to servers to modify text buffers and all that fun stuff.
Read this for the commands servers send.

Unauthenticated Commands

An unauthenticated client might send these commands…

login <string: name> <string: password>
Attempts to authenticate the user.

Authenticated Commands

An authenticated client might send these commands…

logout <String: Parting Message>
Seamlessly logs the user out. Blank parting messages aren't sent.
say <string: message>
Sends a chat message.
