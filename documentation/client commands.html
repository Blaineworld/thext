Thext's Client Commands

This file details the various commands that servers send to clients to update their text displays, show messages, and other actions.
See this document for the commands that clients send to servers.

  1. Authentication
  2. Editing
  3. Session

Authentication

auth.login string:username
…informs the user that they were authenticated and under what username they are logged in.
auth.logout
…informs the user that they were (unauthenticated?).

Editing

edit.set int64:first string:content(1) [string:content(2) ...]
…sets the content of one or more lines.

Session

session.alert string:message
…displays a notification to the user.
session.ban [string:reason]
…closes the connection, telling the user that they were banned from this server and why.
session.kick [string:reason]
…closes the connection, telling the user that they were kicked from this server and why.
