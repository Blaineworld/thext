Thext's Client Commands

This file details the various commands servers send to clients to update their text windows and all that fun stuff.
Read this for the commands clients send.

Universal Commands

Any client should expect these commands…

Alert <string Message>
Alerts the user to something.
Kicked <string Reason>
Alerts the user that they have been kicked.

Unauthenticated Commands

An unauthenticated client should expect these commands…

LoginFailure <string Reason>
Indicates a failure to authenticate the user.

Authenticated Commands

An authenticated client should expect these commands…

Said <string Sender> <string Message>
Displays a chat message.
